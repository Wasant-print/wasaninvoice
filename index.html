🔗 Google Sheets — คู่มือตั้งค่า
  1. เปิด Google Cloud Console → สร้าง Project ใหม่
  2. เปิดใช้ Google Sheets API และ Google Drive API
  3. สร้าง API Key (สำหรับอ่าน) และ OAuth 2.0 Client ID (สำหรับเขียน)
  4. เปิด Google Sheets แล้วแชร์ให้ "Anyone with the link can view"
  5. คัดลอก Sheet ID จาก URL: docs.google.com/spreadsheets/d/[SHEET_ID]/edit
  6. แก้ไขค่า CONFIG.SHEET_ID และ CONFIG.API_KEY ในโค้ด
  7. เพิ่ม sheet ชื่อ users (คอลัมน์: username, password, role, name) และ login_log (username, action, timestamp, ip)
หมายเหตุ: สำหรับการเขียนข้อมูล (สร้าง/แก้ไข/ลบเอกสาร) ต้องตั้งค่า OAuth หรือ Google Apps Script Web API เป็น proxy เพื่อความปลอดภัย
