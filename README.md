# วสันต์ปริ้นเตอร์ — ระบบออกเอกสาร

ระบบออกใบเสนอราคา ใบเสร็จรับเงิน ใบแจ้งหนี้ ใบกำกับภาษี และใบส่งของ
สร้างด้วย Google Apps Script เชื่อมต่อ Google Sheets

---

## ไฟล์ในโปรเจกต์

| ไฟล์ | หน้าที่ |
|------|---------|
| `Code.gs` | Backend — อ่าน/เขียน Google Sheet, ระบบ Login |
| `Index.html` | Frontend — หน้าเว็บแอป UI ขาว-ฟ้า |
| `README.md` | คู่มือนี้ |

---

## ฟีเจอร์

- ✅ ระบบ Login / Logout (Admin & Staff)
- ✅ Admin — สร้าง/แก้ไข/ลบเอกสาร + จัดการผู้ใช้
- ✅ Staff — สร้างและดูเอกสารได้
- ✅ ออกเอกสาร 5 ประเภท
- ✅ ใบเสร็จรับเงิน — ไม่มี VAT ไม่มีเลขภาษี เหมือนใบเสร็จทั่วไป
- ✅ โลโก้และลายเซ็น — บันทึกลง Sheet ใช้ได้ทุกเครื่อง ไม่หาย
- ✅ Export PDF / พิมพ์
- ✅ รองรับมือถือ แท็บเล็ต คอมพิวเตอร์

## วิธีติดตั้ง

**1.** เปิด Google Sheet → **ส่วนขยาย → Apps Script**

**2.** วาง `Code.gs` ลงในไฟล์ Code.gs เดิม กด Ctrl+S

**3.** สร้างไฟล์ใหม่ชื่อ **Index** (HTML) → วาง `Index.html` กด Ctrl+S

**4.** คลิก **ทำให้ใช้งานได้ → ทำให้ใช้งานได้รายการใหม่**
   - ประเภท: เว็บแอป
   - ดำเนินการในฐานะ: ฉัน
   - ผู้เข้าถึง: ทุกคนที่มีบัญชี Google
   - คลิก ทำให้ใช้งานได้ → อนุญาตสิทธิ์

**5.** สร้าง Sheet ใน Google Sheet ชื่อ:
   - `ฐานข้อมูลเอกสาร`
   - `ฐานข้อมูลรายการ`
   
   (Sheet `ตั้งค่าระบบ` และ `ผู้ใช้งาน` สร้างอัตโนมัติ)

## วิธี Push ขึ้น GitHub (ครั้งแรก)

```bash
mkdir wasant-invoice && cd wasant-invoice
# วางไฟล์ Code.gs, Index.html, README.md ลงในโฟลเดอร์นี้

git init
git add .
git commit -m "first commit"
git remote add origin https://github.com/YOUR_USERNAME/wasant-invoice.git
git push -u origin main
```
